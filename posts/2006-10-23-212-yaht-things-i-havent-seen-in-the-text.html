YAHT: Things I haven't seen in the text

2006-10-23 - Magnus Therning

If there’s a type Foo a b, then it’s possible to access a and b in a function definition through the notation f@(x y):

bar :: Foo a b -> Int
bar f@(x y) = ...

Very handy indeed.
